Flint To-Do App

A real-time To-Do application built with Laravel (backend) and React.js (frontend). This app features user authentication, task management with CRUD operations, and real-time updates.

Features

User Authentication: Register, login, and manage sessions using cookies.

Real-Time Task Updates: Tasks can be moved, updated, edited, or deleted in real time on the dashboard.

Notifications: Toast notifications for login, registration, and task actions.

Technologies Used:

Backend: Laravel

Frontend: React.js

Database: PostgreSQL

Cache & Session Management: Redis

Real-Time Events: Pusher
